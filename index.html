Hojas de estilo

Son documentos que definen la presentación visual de un documento HTML. Utilizan CSS (Cascading Style Sheets) para especificar cómo se deben mostrar los elementos en una página web, permitiendo separar el contenido de su diseño.

Sectores

Son patrones utilizados en CSS para seleccionar los elementos HTML a los que se les aplicarán estilos. Existen varios tipos de selectores, como selectores de tipo (por nombre de etiqueta), clase, ID, y atributos, cada uno con su propia sintaxis y propósito.

Listas de declaraciones

Se refieren a un conjunto de reglas CSS que se aplican a un selector específico. Cada regla consiste en una propiedad y un valor, separados por dos puntos, y múltiples declaraciones se separan por punto y coma dentro de llaves.

Section

Es un elemento HTML que representa una sección temática en un documento. Se utiliza para agrupar contenido relacionado y puede incluir encabezados, párrafos y otros elementos. Su uso mejora la estructura semántica del documento.

Hgroup

Este elemento HTML agrupa encabezados relacionados (como h1, h2, etc.) en una sola unidad. Aunque no es tan comúnmente utilizado, ayuda a organizar el contenido en secciones jerárquicas.

Posicionamiento de elementos web

Se refiere a cómo se colocan los elementos en una página web utilizando propiedades CSS como position, top, left, right, bottom, float, y clear. Existen diferentes métodos de posicionamiento: estático, relativo, absoluto y fijo, cada uno con su propio comportamiento en el flujo del documento.

CSS

Cascading Style Sheets (Hojas de Estilo en Cascada) es un lenguaje utilizado para describir la presentación de documentos HTML. Permite definir estilos como colores, fuentes, espaciados y disposiciones, facilitando la creación de interfaces visuales atractivas y consistentes en la web.
